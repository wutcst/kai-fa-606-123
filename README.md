# Starline Valkyrie 横版太空射击游戏

`Starline Valkyrie` 是一个基于 HTML5 Canvas 的横版街机打飞机游戏。项目选择不继续扩展 `world-of-zuul` 文本样例，而是按照实训任务允许的“自选择新的游戏项目内容”方向，完成一个具备完整交互界面和功能逻辑的网页游戏。

## 项目简介

玩家驾驶战机在横向星际战场中移动，系统自动开火并轻微自动索敌。敌机从右侧持续进场，玩家通过走位、连击、拾取补给和释放 BOMB 清屏技能获得更高分数。游戏强调“打开即玩”的街机爽感：高频射击、爆炸反馈、Boss 压迫感、连击成长和霓虹粒子特效。

## 核心功能

- 横版 Canvas 实时游戏循环
- WASD / 方向键移动，移动端支持触控拖动
- 自动开火与轻微自动索敌
- 多类型敌人：轻型机、装甲机、追踪机、炮艇、阶段 Boss
- 子弹、敌弹、碰撞、伤害、生命值和无敌帧
- 分数、击杀数、连击系统
- 掉落物：火力强化、回血、BOMB 充能
- BOMB 清屏技能与大范围冲击波特效
- 暂停、重开、游戏结束状态
- 程序化飞船、敌机、Boss、星空、爆炸和粒子特效
- 自动化单元测试覆盖核心规则和游戏状态机
- 性能优化：背景缓存、批量绘制、粒子数量上限、Retina Canvas 像素上限

## 运行方式

项目无需安装第三方依赖，直接使用浏览器和 Node.js 即可运行。

```bash
npm start
```

启动后访问：

```text
http://127.0.0.1:4173/
```

## 操作说明

| 操作 | 说明 |
| --- | --- |
| WASD / 方向键 | 移动战机 |
| 鼠标 / 触控拖动 | 移动战机 |
| Space | 释放 BOMB 清屏技能 |
| 右下角 BOMB 按钮 | 释放 BOMB |
| P | 暂停 / 继续 |
| R / Enter | 游戏结束或暂停时重新开始 |

## 技术选型

- HTML5 Canvas 2D
- 原生 JavaScript ES Modules
- CSS 自定义界面样式
- Node.js 内置 `node:test` 测试框架
- 无外部贴图库：全部飞船、弹幕、爆炸、星空和 UI 反馈均为程序化绘制

## 开发说明

本项目以“完整图形化游戏体验”为目标，不依赖后端服务。核心逻辑拆分为规则层、游戏状态层和渲染层：

- `rules.js` 保存纯规则函数，便于单元测试。
- `game.js` 负责实体生命周期、碰撞、分数、连击、Boss、BOMB 和掉落。
- `render.js` 只负责 Canvas 绘制，避免把游戏规则写入渲染逻辑。
- `main.js` 负责浏览器输入、窗口缩放、HUD 同步和主循环调度。
